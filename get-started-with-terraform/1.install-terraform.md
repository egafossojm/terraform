# Terraform Installation

### 1. Get installation instructuctions
- Go to [Hashicorp Dev website](https://developer.hashicorp.com/terraform/install).
- We are going to install using packet manager (This way we could easily update terraform)
- Select your Operating System and get installation instructions

### 2. Install on Centos (My case)
- Run commands for your OS:
```bash
sudo yum install -y yum-utils
sudo yum-config-manager --add-repo https://rpm.releases.hashicorp.com/RHEL/hashicorp.repo
sudo yum -y install terraform
```
### 3. Enable Tab completion(far bash or zsh)
- Make sure the `.bashrc` or `.zshrc` file exists in the user home directory
```bash
$ ls -al ~| grep .bashrc
-rw-r--r--.  1 user  user          8349 Nov 13 09:44 .bashrc
```
- If not, create it 
- Then installl the auto-complete package
```bash
$ terraform install-autocomplete
```

### 4. Enable terraform extension in VSCode
- This will enable `syntax highlighting` and `auto completion` for terraform.
- Click on **Extensions** button and search for **terraform**
- Install `Hashicorp Terraform` extension (The one verified by Hashicorp)
